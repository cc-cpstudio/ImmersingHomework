# ImemrsingHomework 0.1.0.0-beta

发布日期: 2026-05-24

## 📝 概述

这是 ImmersingHomework 的首个测试版本，提供了基础的班级大屏作业展示功能。

## ✨ 新功能

### 核心功能
- 科目管理 - 支持添加、编辑、删除科目
- 作业记录 - 按科目记录和展示作业内容
- 标签系统 - 使用标签分类和筛选作业
- 作业图片生成 - 支持生成作业图片

### 用户界面
- 欢迎向导 - 首次启动时的初始化配置
- 主窗口 - 全屏作业展示界面
- 浮窗按钮 - 主窗口最小化时显示快速访问按钮
- 设置窗口 - 用户从此处修改软件设置
